# Contributing to the Pi Coding Agent Guide

Thanks for your interest in improving this guide! Here's how you can help:

## How to Contribute

1. **Fork** this repository
2. **Create a branch** for your changes
3. **Make your changes** to the relevant Markdown files
4. **Submit a Pull Request**

## Guidelines

- Keep it beginner-friendly — this is a guide, not a reference manual
- Use clear, concise language
- Include practical examples where possible
- Test any commands or code snippets before submitting
- Update the Table of Contents if you add new sections

## What Needs Help

- Translations to other languages
- More example extensions and skills
- Screenshots and GIFs of Pi in action
- Tips and tricks for specific workflows
- Video tutorial links

Thanks! 🥧
